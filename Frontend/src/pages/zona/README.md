# Modulo Zona

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `zona`.

## Componentes Principales
- `ZonaList.jsx`: Muestra la lista de registros.
- `ZonaForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/zona`: Vista de lista
- `/zona/new`: Crear nuevo
- `/zona/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`zonaService.js`)
