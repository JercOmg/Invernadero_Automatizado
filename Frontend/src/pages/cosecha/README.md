# Modulo Cosecha

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `cosecha`.

## Componentes Principales
- `CosechaList.jsx`: Muestra la lista de registros.
- `CosechaForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/cosecha`: Vista de lista
- `/cosecha/new`: Crear nuevo
- `/cosecha/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`cosechaService.js`)
