# Modulo Lectura_sensor

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `lectura_sensor`.

## Componentes Principales
- `Lectura_sensorList.jsx`: Muestra la lista de registros.
- `Lectura_sensorForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/lectura_sensor`: Vista de lista
- `/lectura_sensor/new`: Crear nuevo
- `/lectura_sensor/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`lectura_sensorService.js`)
