# Modulo Dashboard

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `dashboard`.

## Componentes Principales
- `DashboardList.jsx`: Muestra la lista de registros.
- `DashboardForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/dashboard`: Vista de lista
- `/dashboard/new`: Crear nuevo
- `/dashboard/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`dashboardService.js`)
