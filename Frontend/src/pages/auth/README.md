# Modulo Auth

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `auth`.

## Componentes Principales
- `AuthList.jsx`: Muestra la lista de registros.
- `AuthForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/auth`: Vista de lista
- `/auth/new`: Crear nuevo
- `/auth/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`authService.js`)
