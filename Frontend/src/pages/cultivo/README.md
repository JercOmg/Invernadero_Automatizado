# Modulo Cultivo

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `cultivo`.

## Componentes Principales
- `CultivoList.jsx`: Muestra la lista de registros.
- `CultivoForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/cultivo`: Vista de lista
- `/cultivo/new`: Crear nuevo
- `/cultivo/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`cultivoService.js`)
