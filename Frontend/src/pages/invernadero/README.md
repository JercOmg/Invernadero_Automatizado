# Modulo Invernadero

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `invernadero`.

## Componentes Principales
- `InvernaderoList.jsx`: Muestra la lista de registros.
- `InvernaderoForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/invernadero`: Vista de lista
- `/invernadero/new`: Crear nuevo
- `/invernadero/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`invernaderoService.js`)
