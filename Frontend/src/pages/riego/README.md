# Modulo Riego

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `riego`.

## Componentes Principales
- `RiegoList.jsx`: Muestra la lista de registros.
- `RiegoForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/riego`: Vista de lista
- `/riego/new`: Crear nuevo
- `/riego/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`riegoService.js`)
