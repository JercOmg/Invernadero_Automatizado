# Modulo Insumo

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `insumo`.

## Componentes Principales
- `InsumoList.jsx`: Muestra la lista de registros.
- `InsumoForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/insumo`: Vista de lista
- `/insumo/new`: Crear nuevo
- `/insumo/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`insumoService.js`)
