# Modulo Sensor

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `sensor`.

## Componentes Principales
- `SensorList.jsx`: Muestra la lista de registros.
- `SensorForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/sensor`: Vista de lista
- `/sensor/new`: Crear nuevo
- `/sensor/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`sensorService.js`)
