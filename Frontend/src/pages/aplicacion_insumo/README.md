# Modulo Aplicacion_insumo

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `aplicacion_insumo`.

## Componentes Principales
- `Aplicacion_insumoList.jsx`: Muestra la lista de registros.
- `Aplicacion_insumoForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/aplicacion_insumo`: Vista de lista
- `/aplicacion_insumo/new`: Crear nuevo
- `/aplicacion_insumo/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`aplicacion_insumoService.js`)
