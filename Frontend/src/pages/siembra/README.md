# Modulo Siembra

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `siembra`.

## Componentes Principales
- `SiembraList.jsx`: Muestra la lista de registros.
- `SiembraForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/siembra`: Vista de lista
- `/siembra/new`: Crear nuevo
- `/siembra/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`siembraService.js`)
