# Modulo Alerta

## Descripcion
Este modulo gestiona las vistas y operaciones relacionadas con `alerta`.

## Componentes Principales
- `AlertaList.jsx`: Muestra la lista de registros.
- `AlertaForm.jsx`: Formulario para crear y editar registros.

## Rutas
- `/alerta`: Vista de lista
- `/alerta/new`: Crear nuevo
- `/alerta/:id`: Editar existente

## Dependencias
- `react`, `react-router-dom`
- `react-i18next`
- Servicios del API (`alertaService.js`)
